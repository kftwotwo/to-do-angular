# To do with angular

 This is a template for angular 2, 1/17/2017

 By [Kevin Finley](http://www.kfinley.com)

## Description

Able to add tasks for a to-do list.

## Setup/Installation Requirements

Clone the repository and `cd` into your directory:
```
$ npm install
$ bower install
$ gulp build
$ gulp serve
```
